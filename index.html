      3 Paste the SMS profile values

      In SMS, extract the cross-section profiles:

      1. Activate the observation cross-sections coverage (Existing or Proposed).
      2. Top menu Display → Plot Wizard.
      3. Choose plot type Observation Profile, then Next.
      4. Confirm the coverage is the observation cross-sections, and that all arcs are selected / active.
      5. Under Data sets, switch from Active to Specified, pick the mesh, then check the terrain Z (ground) plus each flood water-surface elevation you need (2-year, 100-year, 500-year, …).
      6. Under Time steps, choose Specified and check only the last time step of the simulation.
      7. Click Finish — the profile plot appears.
      8. Right-click the plot → View Values → Ctrl+A, Ctrl+C.
      9. Paste below.
